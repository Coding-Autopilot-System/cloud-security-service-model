# Sentinel (Stub)

## Detection engineering workflow
1. Identify detection requirement.
2. Author analytic rule in a test workspace.
3. Peer review and validate logic.
4. Deploy to production via pipeline.
5. Review alert quality monthly.

## Tuning cadence
- Weekly review of false positives.
- Monthly review of detection coverage.
- Quarterly review aligned to threat landscape updates.
