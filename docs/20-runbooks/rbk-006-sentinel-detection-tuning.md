# RBK-006 Sentinel Detection Tuning

## Purpose
Maintain signal quality and reduce alert noise.

## Steps
1. Review alert volume and false positive rate.
2. Update analytic rules and thresholds.
3. Validate detections in a test workspace.
4. Deploy changes via pipeline.
5. Monitor impact for 2 weeks.

## Evidence
- Tuning changeset
- Validation results
- Post-change metrics
