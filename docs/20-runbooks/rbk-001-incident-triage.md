# RBK-001 Incident Triage

## Purpose
Provide a consistent approach to triage security alerts.

## Steps
1. Validate alert source and confidence.
2. Determine severity using the SEV model.
3. Assign incident commander for SEV1/2.
4. Open incident record and begin evidence capture.
5. Execute containment or escalation as required.

## Evidence
- Alert details and timestamps
- Initial triage notes
- Assigned owner
